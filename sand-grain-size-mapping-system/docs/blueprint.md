# **App Name**: PixelGrain Analyzer

## Core Features:

- Loading Screen: Display the 'PixelGrain' logo during the app loading phase.
- Location Permission & GPS: Prompt the user for location permission and retrieve GPS coordinates.
- Sample Image Input: Allow users to upload a sand sample image via camera or file upload, displaying a preview.
- Parameter Input: Input for 'Coin Currency,' 'Coin in Picture,' and 'Location on Beach' to define measurement conditions.
- Sand Analysis: Utilize the local ML model to analyze the image and determine grain size, density, and coarseness. Location parameter, and selected coin parameter will be passed to the model as tool.
- Analysis Loading Screen: Animation during the processing.
- Analysis Summary: Display a summary of the analysis results, including grain size, density, coarseness, GPS location, and coin details.

## Style Guidelines:

- Primary color: Dark blue-purple (#301934) to give a tech-focused and modern feel.
- Background color: Very dark gray/black (#000000) to provide contrast and a sleek appearance.
- Accent color: Electric purple (#8A2BE2) to highlight important data and interactive elements.
- Body and headline font: 'Roboto Mono' monospace font for a technical and precise look.
- Use sharp, geometric icons with a neon purple glow effect for a futuristic aesthetic.
- Design a layout optimized for the 8x5.3 cm LCD display, prioritizing essential information and clear navigation.
- Use simple loading animations with a digital glitch effect to indicate processing.
